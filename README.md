# Stock Price Prediction using Machine Learning

A Machine Learning project that predicts stock prices using historical stock market data and predictive modeling techniques in Python.  
This project focuses on data analysis, visualization, preprocessing, and building prediction models for stock market trends.

---

## 📌 Project Overview

The objective of this project is to analyze historical stock market data and develop a machine learning model capable of predicting future stock prices.

The project includes:
- Data preprocessing
- Exploratory data analysis
- Stock trend visualization
- Model training and prediction
- Performance evaluation

---

## 🛠️ Technologies Used

- Python 3
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📂 Dataset

The dataset contains historical stock market data including:
- Opening Price
- Closing Price
- High and Low Prices
- Volume
- Date-wise Stock Information

The model uses this data to identify patterns and predict future stock prices.

## 🚀 How to Run

1. Clone the repository

```text
git clone https://github.com/your-username/stock-price-prediction.git
```

2. Open the project folder

3. Install required libraries

```text
pip install pandas numpy matplotlib scikit-learn
```

4. Run the Jupyter Notebook

```text
jupyter notebook
```
